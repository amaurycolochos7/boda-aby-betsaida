# Instrucciones: Música de Fondo

## Archivo Requerido

Necesitas colocar tu archivo de música en esta carpeta con el nombre **exacto**:

```
background.mp3
```

## Especificaciones del Archivo

- **Formato**: MP3 (compatible con todos los navegadores)
- **Duración recomendada**: 2-4 minutos (se reproducirá en loop automáticamente)
- **Tamaño máximo recomendado**: 5-10 MB para carga rápida
- **Tipo**: Instrumental o canción romántica (sugerido)

## Cómo Agregar el Archivo

1. Descarga tu canción favorita en formato MP3
2. Renómbrala a: `background.mp3`
3. Cópiala a esta carpeta: `c:\Users\Amaury\.gemini\antigravity\scratch\boda-abidan-betsaida\music\`

## Funcionamiento

✅ La música iniciará automáticamente cuando el usuario haga clic en "ENTRAR"
✅ Se reproducirá en loop continuo
✅ Volumen configurado al 50% para no ser invasivo
✅ Sin controles visuales (como solicitaste)

## Si No Tienes Archivo Aún

Si no tienes el archivo de música todavía:
- La página funcionará normalmente sin errores
- Simplemente no habrá música hasta que agregues el archivo
- Puedes agregarlo en cualquier momento sin modificar código

## Probar

1. Abre `http://localhost:8080/`
2. Haz clic en "ENTRAR"
3. La música debería empezar a reproducirse automáticamente
